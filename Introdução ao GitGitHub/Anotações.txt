O git bash é um terminal extendido para otimizaro uso do git.

Link para download
https://docs.github.com/pt/desktop/installing-and-configuring-github-desktop/installing-and-authenticating-to-github-desktop/installing-github-desktop

Comandos:

git clone "link https repositoria do git hub".

cd -- entrar nas pastas 

ls -- listar as pastas 

git status -- Indica a siruação do repositório.

git add . -- Enviar a pasta pro controle de versão adicionando os arquivos com versões novas.

git commit -m "" -- Cria o commit para o arquivo de forma local.

git push origin main -- Envia os commits locais para a nuvem.
